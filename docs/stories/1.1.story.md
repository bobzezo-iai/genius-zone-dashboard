## Story

**Como** operador do Genius Zone Dashboard,
**Quero** uma rota `/c/{slug}` que carrega o perfil psicométrico pré-curado de um cliente,
**Para que** cada cliente tenha um link exclusivo com seu dashboard personalizado.

---

## Acceptance Criteria

- [ ] **AC1:** URL `/c/jivan-pramod` carrega `index.html` via Vercel rewrite
- [ ] **AC2:** `client-loader.js` extrai slug da URL (`/c/{slug}`) e chama `GET /api/client-profile?slug={slug}`
- [ ] **AC3:** API `client-profile.js` lê `grimoires/{slug}/profile.json` e retorna JSON
- [ ] **AC4:** Slug validado com regex `/^[a-z0-9-]{2,60}$/` — rejeita qualquer path traversal (`..`, `/`, `\`)
- [ ] **AC5:** Rate limiting na API: 10 requests/min/IP (usar `rate-limiter.js` existente)
- [ ] **AC6:** Se `profile.json` não existe, retorna `{ status: 'pending', message: 'Perfil em preparação' }` com HTTP 404
- [ ] **AC7:** Frontend mostra mensagem amigável "Seu perfil está sendo preparado" quando status=pending
- [ ] **AC8:** Dados retornados são salvos no localStorage com chave `psicometria_data_v2_client_{slug}`
- [ ] **AC9:** `vercel.json` contém rewrite: `{ "source": "/c/:slug", "destination": "/index.html" }`
- [ ] **AC10:** `index.html` inclui `<script src="js/client-loader.js"></script>` antes dos outros scripts
