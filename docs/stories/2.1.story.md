## Story

**Como** cliente premium que acessou seu link exclusivo pela primeira vez,
**Quero** que a Aurora me guie por um assessment conversacional com perguntas sobre meu perfil,
**Para que** meu mapa de genialidade seja gerado automaticamente sem precisar de documentos.

---

## Acceptance Criteria

### Assessment Mode
- [ ] **AC1:** Ao acessar `/c/{slug}` sem profile.json, Aurora abre automaticamente em modo assessment (panel aberto, FAB visível)
- [ ] **AC2:** Aurora apresenta boas-vindas: "Olá! Sou a Aurora... vou guiar você por um assessment de ~30 min para mapear sua Zona de Genialidade"
- [ ] **AC3:** Perguntas carregadas de `data/assessment-questions.json` — motor 100% frontend, ZERO chamadas à API durante o assessment
- [ ] **AC4:** Perguntas multiple choice renderizadas como opções numeradas no chat (cliente digita número ou texto)
- [ ] **AC5:** Perguntas open-ended aceitam texto livre
- [ ] **AC6:** Indicador de progresso: "Seção 2 de 5 — Atividades e Energia (pergunta 3 de 12)"
- [ ] **AC7:** Progresso salvo no localStorage a cada seção completada (chave: `assessment_progress_{slug}`)
- [ ] **AC8:** Se cliente fecha browser e volta, Aurora retoma de onde parou: "Bem-vindo de volta! Você parou na Seção 3..."

### Âncoras de Calibração
- [ ] **AC9:** Antes do questionário, Aurora pergunta: "Você já tem resultados de outros testes de autoconhecimento?"
- [ ] **AC10:** Se SIM, apresenta lista de âncoras conhecidas uma a uma (Eneagrama, DISC, MBTI, Numerologia, Signo, Human Design, Inteligências Múltiplas, Traços de Caráter, Missão de Vida)
- [ ] **AC11:** Cliente preenche apenas as que tem — pode pular qualquer uma
- [ ] **AC12:** Se NÃO, segue direto para as 43 perguntas

### Campo Aberto
- [ ] **AC13:** Após âncoras, Aurora pergunta: "Tem mais alguma coisa sobre você que acha relevante? Outros testes, avaliações, descobertas pessoais..."
- [ ] **AC14:** Aceita texto livre até 5000 chars, salvo como `additional_context`
- [ ] **AC15:** Cliente pode pular (responder "não" ou "pular")

### Refazer Assessment
- [ ] **AC16:** Cliente com profile.json existente vê botão "Refazer Assessment" no dashboard
- [ ] **AC17:** Ao clicar, limpa localStorage (profile + assessment progress) e abre Aurora em assessment mode
