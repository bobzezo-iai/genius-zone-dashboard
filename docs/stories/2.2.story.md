# Story 2.2 — Geração Automática de Profile.json via Assessment

**Epic:** Multi-Cliente Premium com Aurora Contínua
**Spec:** SPEC-GZD-002
**Refs:** FR-8, NFR-6, DATA-1
**Status:** Draft

## Story

**Como** cliente premium que completou o assessment guiado pela Aurora,
**Quero** que meu profile.json seja gerado automaticamente a partir das minhas respostas,
**Para que** meu dashboard de genialidade apareça imediatamente sem intervenção do operador.

---

## Acceptance Criteria

- [ ] **AC1:** Ao completar a última pergunta (seção 5), Aurora compila todas as respostas + âncoras + campo aberto em texto estruturado
- [ ] **AC2:** Texto compilado segue formato: `# Assessment Zona de Genialidade — {nome}\n## Âncoras\n...\n## Seção 1\n...`
- [ ] **AC3:** Aurora mostra mensagem: "Analisando seu perfil nos 7 frameworks... isso leva de 3 a 5 minutos"
- [ ] **AC4:** POST `/api/analyze` com texto compilado (reutiliza API existente, zero mudança no backend)
- [ ] **AC5:** profile.json retornado é salvo no localStorage com chave `psicometria_data_v2_client_{slug}`
- [ ] **AC6:** Dashboard renderiza automaticamente via `PsicoRenderer.render(data)`
- [ ] **AC7:** Aurora transiciona de modo assessment para modo mentora: "Seu mapa de genialidade está pronto! Me pergunte qualquer coisa sobre seu perfil..."
- [ ] **AC8:** Dados de assessment parcial limpos do localStorage após profile.json gerado com sucesso (cleanup)
- [ ] **AC9:** Se /api/analyze falhar, Aurora mostra erro amigável e permite tentar novamente
- [ ] **AC10:** Assessment completo em ≤35 minutos incluindo âncoras e campo aberto (NFR-6)

---

## Dev Notes

- `/api/analyze.js` NÃO precisa de mudança — aceita texto livre, retorna profile.json
- O texto compilado deve incluir âncoras no topo para que o Sonnet use como contexto de calibração
- Auth headers: usar mesma lógica de `psicometria-upload.js` (Auth.authHeaders())
- Em caso de erro 429 (quota), mostrar mensagem com data de reset
- Custo por assessment: 1 chamada Sonnet (~$0.03-0.10 dependendo do tamanho)
- Após geração, Aurora.init() recria o chat em modo mentora com saudação personalizada
